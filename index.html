Maximus

Overview

MAXIMUS is automation module for MING project that is scheduled to run once a day. It reads the records in excel file located in the FTP server, validates each records in the file and determines whether the file is valid for processing or not. If valid, it loads the record data to the respective tables otherwise dumped the filename to the error table. The module also sends the email to the case manager clarifying whether the sent file is valid or not. If the file is invalid, it sends the details of the error. Integration of this module minimized the ETL (Extract, Tranform, Load) work by 80%.
